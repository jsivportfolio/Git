GIT BASH
- Mac comes with Git Bash pre-installed, buti if you're working on windows? 

1. Download link for GitBash: https://gitforwindows.org/
2. Open installer > Yes > Run through the wizard > Agree to License > Click Next and Keep default options > Finish
3. Make Git Bash the default terminal in the VS CODE IDE
    - View > Command Palette... > Select Default Profile > Git Bash
    - if it doesn't appear right away, then close VSCode and reopen it

TERMINAL MAY APPEAR INTIMIDATING BUT IT BECOMES VERY USEFUL WHEN LEARNED.
- greater speed, control and power of dev tools, it's easier and faster to do common things

BASH => Bourne Again SHell
- CLI for UNIX based systems

Commands: 
- `cd <path>`: navigate to directory
- `cd ..`: navigate back a directory
- `cd ~`: navigate to the root directory
- `ls`: list all items in a particular directory
- cycle through command history by pressing up && down arrow keys
- `ctrl + a`: go to beginning of command line
- `ctrl + e`: go to end of command line
- `ctrl + u`: clear an entire command line
- `mkdir <FileName>`: create new folder
- `rm <FileName>`: remove/delete file
- `rm *`: removes all files in path
- `pwd`: tells entire path and current directory
- `rm -r <directorty/>`: removes directory
- `ls -a`: gives list of all folders in current directory (even hidden files)
- `git rm --cached -r .`: removes everything from the staging area
